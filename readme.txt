SASS ayuda porque tiene herramientas de anidación y varias herramientas más para generar un css de forma
más fácil.
Podemos hacer bucles para generar css para las diferentes filas de una tabla por ejemplo ... etc
La opcion --watch te ayuda para poder actualizar el fichero sass, te lo actualiza solo.

SASS me parece una herramienta util para el desarrollo de css de forma más ordenada y más rápida, si tienes
40 filas, con SASS se puede hacer mucho más fácil y rápida que si tienes que hacer el css a mano.
